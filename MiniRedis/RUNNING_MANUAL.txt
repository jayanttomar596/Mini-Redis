Prerequisites:
Make sure your terminal is open in the MiniRedis folder. If not, enter the command:
Bash - cd MiniRedis



1. Compile (Only once or after code changes)
Run this single command in your project folder to build both the server and the client:
Bash - make

(If you ever need to clear out the compiled files to start fresh, simply run make clean.)




2. Start Master Server (Terminal 1)
Bash - ./app

(Alternatively, you can just run make run to compile and start the master server in one step.)

👉 Output:
Server started on port 8080
👉 Keep this terminal OPEN




3. Start Slave Server (Terminal 2 - Optional)
To test your master-slave replication, open a second terminal, ensure you are in the MiniRedis folder, and run:

Bash - ./app slave 8080
👉 Keep this terminal OPEN




4. Start Client (Terminal 3)
Open a third terminal, ensure you are in the MiniRedis folder, and run:

Bash
./client
👉 You’ll see:
>>



5. Run Commands (Terminal 3)
Test your fast key-value store with the following commands:

Plaintext
SET name Jayant
GET name
DEL name
GET name
